## Description:
Dự án webapp ứng dụng AI (trí tuệ nhân tạo) hỗ trợ công việc văn phòng. Xây dựng các công cụ nhỏ (tools) để hỗ trợ công việc văn phòng, hỗ trợ lập trình viên.

## Các tính năng:
- Tạo CV từ thông tin cá nhân.
- Công cụ format code (python, TSX, JSX, HTML, CSS, ...)
- ORC từ ảnh và PDF.

## Frontend:
- Sử dụng NextJS
- Sử dụng TailwindCSS
- Sử dụng Shadcn UI -> dùng để tạo các component UI
- Sử dụng TypeScript
- Sử dụng Zustand -> dùng để quản lý state
- Sử dụng Axios -> dùng để gọi API
- Sử dụng lucide icons -> dùng để hiển thị icon

## Backend:
- Sử dụng FastAPI
- Sử dụng Python
- Sử dụng Uvicorn
- Sử dụng SQLAlchemy -> dùng để kết nối với database
- Sử dụng Alembic -> dùng để quản lý migration
- Sử dụng Pydantic -> dùng để validate data
- Sử dụng PyJWT -> dùng để xác thực token
- Sử dụng python-multipart -> dùng để upload file
- Sử dụng asyncpg -> dùng để kết nối với database không đồng bộ
- Sử dụng FastAPI Mail -> dùng để gửi mail

## Database:
- Sử dụng PostgreSQL
